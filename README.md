# Work-Study-Project
此專案紀錄了我在清華大學工業工程系所修習Work Study做的相關專案或小Project

## 健康檢查排程
此專案為本課程的一個期中小作業。基於分數占比不高，也不是台灣內大家認為很主流且重要的領域，大部分對這個作業的想法都是得過且過，要馬提出很爛的方式，要馬抄考古。但基於鄙人對此功課富含興趣，因此提出了一些想法來進行，也確實得到了相較於考古或同學們的作業更好的結果。

### 加工程序
將健康檢查視為一道道沒有前後相關的加工程序後即可以應用各種加工方法去操作，例如：Line Balance或甘特圖。基於此方法，我們將好幾個 Station 基於時間與人數分組，並讓在 Station 中輪轉。

### Layout設計
為了減少組間輪轉所消耗的時間，在有限的空間內將每個組分成相鄰的環狀。

最後的結果的時間使用效率為84%，且只使用了20位醫生與4台X光車。

## 元興塑膠企業社的工廠研究
此作業是我們的期末報告，我們要去研究一家工廠在動作研究以及時間研究的部分，並提供優化的可能性。我們研究的是一家南部傳統專門做塑膠加工的工廠。並提出了一些佈局上的改善可能。

### Product Cycle time
了解四個不同產品的生產週期，與員工的工作習慣，包含兩個射出成型和吹氣成型的產品。

### Factory layout
透過 SLP Relationship 了解各個工序之間的關係來重新規劃工廠的布局。在 Injection Molding 的工序上減少了39.4% 的時間，而 Blow Molding 減少 46.1% 的時間。

## 個人心得
Work Study 是工工系中非常重要的課，但比較難過的是感覺沒什麼人在乎，那些班上前幾名的也是如此。可能是因為這堂課不像程式語言等主流課程或是數學類等基礎課程，因此很多人都選擇刷刷考古抄抄家傳，最後期末的成績還是很高。雖然最後一週作息混亂導致期末大爆炸所以成績也沒有到很好看，但我覺得這些內容是對我們有幫助且很好玩的，也期待可以在未來碰到更真實的產業。
